# Import Data using Sqoop

This project demonstrates how to transfer data from a relational database into Hadoop using Sqoop.

## Technologies Used
- Python
- Pandas
- SQLite
- Google Colab
- Hadoop
- Sqoop

## Steps
1. Import libraries
2. Create a relational database using SQLite
3. Insert customer data
4. Read data from the database
5. Transfer data using Sqoop
